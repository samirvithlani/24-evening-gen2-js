window object
- window object is the browser window
- window object is the top level object in the browser
- window object is the global object in client side javascript
- window object is the parent object of the browser

document object
- document object is the webpage
- document object is the child object of the window object
- document object is the global object in client side javascript
- document object is the parent object of the webpage
